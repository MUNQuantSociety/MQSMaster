# MQS Trading System - Workflow Diagrams

This folder contains Mermaid.js diagrams documenting the architecture and data flows of the MQS Trading System.

## Diagrams

1. **system-architecture.md** - High-level system architecture overview
2. **live-trading-flow.md** - Live trading execution workflow
3. **backtest-flow.md** - Backtesting engine workflow
4. **data-pipeline.md** - Market data ingestion and backfill pipeline
5. **portfolio-strategy-flow.md** - Strategy signal generation and execution
6. **capital-management.md** - Risk management and capital allocation

## How to View

These diagrams use Mermaid.js syntax. You can view them:
- In GitHub (renders automatically)
- In VS Code with Mermaid extension
- At [mermaid.live](https://mermaid.live)
- In any Markdown viewer with Mermaid support
